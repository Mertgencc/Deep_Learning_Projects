### 🌍 Global Oil Price Analysis with Deep Learning Regression

#### 📌 Project Overview

This project aims to analyze and predict the impact of global geopolitical events on oil prices using deep learning techniques.  
A neural network-based regression model is developed to learn patterns from historical data and make predictions.

---

#### 🎯 Objective

- Analyze the relationship between global events and oil prices  
- Build a deep learning-based regression model  
- Predict oil price movements using historical data  

---

#### 📊 Dataset

The dataset used in this project is sourced from Kaggle:

https://www.kaggle.com/datasets/zkskhurram/global-petrol-prices-impact-of-2026-us-iran-war

---

#### 🛠️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib  
- Jupyter Notebook  

---

#### 🧠 Model Architecture

A deep neural network regression model was used:

- Fully connected (Dense) layers  
- ReLU activation function  
- Adam optimizer  
- Mean Squared Error (MSE) loss function  

---

#### 📈 Model Evaluation

The model performance was evaluated using:

- R² Score  
- RMSE (Root Mean Squared Error)  
- Loss / Validation Loss curves  

---

#### ⚙️ Training Process

- Dataset split into training and testing sets  
- Feature scaling applied using StandardScaler  
- EarlyStopping used to prevent overfitting  
- Model trained using epoch-based learning  

---

#### 📌 Results

The model successfully learned the relationship between input features and oil price variations.  
Evaluation metrics show that the model produces meaningful regression predictions.
