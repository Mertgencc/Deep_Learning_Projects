## 🌍 Oil Price Analysis with Regression

### 📌 Project Overview

- In this project, I analyzed the relationship between global conflicts (such as the Iran–USA tension) and oil prices using a regression model. The goal was to see how such events might affect price movements and whether a machine learning model can capture this relationship

### ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- TensorFlow / Keras

### 🧠 What I Did
- Collected and prepared the dataset
- Built a regression model
- Trained the model and observed the learning process
- Evaluated performance using loss and R² metrics
- Interpreted the results

### 📊 Results & Observations

- The model was able to learn patterns in the data to some extent, and loss values decreased during training. The R² score indicated a reasonable fit to the dataset.
- However, since the dataset is relatively small, the model may not generalize well to real-world scenarios. There is also a possibility of overfitting.


### ⚠️ Limitations

- Limited dataset size
- External factors affecting oil prices were not fully included
- Results should be interpreted carefully

### 🚀 Future Improvements

- Use a larger and more diverse dataset
- Include additional variables (economic indicators, geopolitical factors)
- Try different models and optimization techniques
